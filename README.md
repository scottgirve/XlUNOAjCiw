# Python汽车租赁管理系统

## 核心功能
- 首页大屏展示
- 汽车展览推荐和数据可视化
- 汽车品牌管理
- 车辆信息管理
- 车辆查询
- 租车和还车操作
- 用户评论和打分
- 角色权限管理
- 评论内容管理
- 用户信息管理
- 公告管理

## 适用场景
- 汽车租赁公司
- 车辆租赁平台
- 汽车销售企业
- 个人租车服务

## 技术特点
- 基于Django框架构建，确保系统稳定性
- 采用MySQL数据库存储数据，确保数据安全
- 利用Vue.js实现前端界面，提供良好的用户体验
